python-montage - A python Montage wrapper

Website: http://astrofrog.github.com/python-montage/
License: MIT
Description:
Python-montage is a pure Python module that provides a Python API to the
Montage Astronomical Image Mosaic Engine, including both functions to access
individual Montage commands, and high-level functions to facilitate mosaicking
and reprojecting.
